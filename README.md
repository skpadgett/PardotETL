# PardotETL

This ETL is used to extract the following data from SalesForce:

* "ListMembership"
* "Prospect"
* "ProspectAccount"
* "Visitor"
* "VisitorActivity"
* "Campaign"
* "Form"
* "Tag"
* "TagObject"
* "Opportunity"
* "EmailClick"
* "List"
* "Account"
